## 🌐 Deployment to Netlify (30 minutes)

### Step 1: Create GitHub Repository
```bash
# In your terminal (from /Users/eletu/tems-aura):
git add .
git commit -m "🎉 Initial commit: TEMS AURA luxury fashion website"
```

Then:
1. Go to https://github.com/new
2. Repository name: `tems-aura`
3. Make it **Public** (for free Netlify)
4. Don't initialize with README
5. Click "Create repository"

```bash
# Copy the commands GitHub shows, something like:
git remote add origin https://github.com/YOUR_USERNAME/tems-aura.git
git branch -M main
git push -u origin main
```

### Step 2: Deploy to Netlify
1. Go to https://netlify.com (sign up with GitHub - it's FREE!)
2. Click "Add new site" → "Import an existing project"
3. Choose "GitHub"
4. Select your `tems-aura` repository
5. Click "Deploy site" (settings auto-detected from netlify.toml)
6. Wait 2-3 minutes ⏳
7. 🎉 Your site is live! You'll get a URL like: `tems-aura-xyz123.netlify.app`

### Step 3: Enable CMS (Required!)
1. In Netlify dashboard → Click "Identity"
2. Click "Enable Identity"
3. Go to Settings → Registration → Change to **"Invite only"**
4. Click "Invite users"
5. Enter your email: `tems_aura@gmail.com`
6. Check email, accept invitation, set password
7. Now you can access `/admin` to manage content!

### Step 4: Enable Forms
- Already enabled automatically! ✅
- Go to "Forms" tab in Netlify to see submissions

## 🆘 Quick Troubleshooting

### "Site not loading"
- Clear browser cache (Cmd+Shift+R)
- Check Netlify deployment status
- Verify domain DNS settings

### "Can't login to /admin"
- Ensure Netlify Identity is enabled
- Check you accepted invitation email
- Try password reset
- Use incognito mode

### "Form not submitting"
- Check Netlify Forms dashboard
- Verify form has data-netlify attribute
- Check spam folder for notifications
- Test in different browser

### "Images not showing"
- Check file path is correct
- Verify image uploaded to /uploads
- Clear browser cache
- Re-deploy site
