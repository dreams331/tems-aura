# 🔐 Quick CMS Access Guide

## Your CMS Admin URL
```
https://[your-vercel-domain]/admin
```

## First-Time Setup (5 minutes)

### 1. Create Netlify Site (For Auth Only)
- Go to: https://app.netlify.com
- Connect your GitHub repo: `dreams331/tems-aura`
- Deploy (it won't interfere with Vercel)

### 2. Enable Identity & Git Gateway
- In Netlify → Identity tab
- Click "Enable Identity"
- Set registration to "Invite only"
- Enable "Git Gateway" under Services

### 3. Invite Yourself
- In Identity → Click "Invite users"
- Email: `tems_aura@gmail.com`
- Check email → Accept invite → Set password

### 4. Login to CMS
- Visit: `https://[your-vercel-site]/admin`
- Login with your email/password
- Start managing content! ✨

## How It Works

1. **You edit content** in the CMS dashboard
2. **Changes are committed** to your GitHub repository
3. **Vercel auto-deploys** the updates
4. **Your live site updates** in ~1 minute
